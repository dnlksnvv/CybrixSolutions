Как запустить

        
    
          
  1. Запусти сервер из каталога CS-VoiceAgent:
            uvicorn app.main:app --reload --host 127.0.0.1 --port 8008
            затем открой в браузере http://127.0.0.1:8008/ (порт можно другой).
          
    2. 
          
  2. В Settings сохрани DashScope key и при необходимости блок «Голосовой чат → LLM» (модель, system prompt, отдельный API key).
    3. 
          
  3. Ниже выбери модель/язык ASR и голос TTS именно для этого сценария.
    4. 
          
  4. Нажми «Начать сценарий» — браузер спросит микрофон; говори предложениями с . ! ? в конце.
    5. 
          
  5. Справа в чате увидишь стрим ответа LLM и услышишь озвучку. «Остановить» — завершить STT и TTS.
    6. 
        

        

          В LLM текст уходит после тишины STT: нет новых partial/final дольше заданного числа миллисекунд — можно «договорить» без нового запроса на каждый сегмент.
          Ответ стримится в чат; в TTS — по готовым предложениям в потоке (. ! ? и перенос строки). При новой речи можно прервать генерацию и/или обнулить очередь звука (контекст в чате сохраняется).
